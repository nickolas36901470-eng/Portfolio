# Portfolio Website Agent

## Purpose
This agent helps build a static portfolio website using HTML and CSS for a college student with programming experience in Java and Python.

## Goals
- Create a simple, responsive static portfolio site.
- Include sections for About, Skills, Projects, Education, and Contact.
- Highlight the student's Java and Python knowledge.
- Use clean, modern HTML and CSS only.

## Recommended Structure
- `index.html` for the main content
- `styles.css` for styling
- Optional `assets/` folder for images or icons

## Content Guidelines
- Write concise, friendly copy suitable for a college student.
- Emphasize project experience and academic background.
- Include a skills list that mentions Java, Python, HTML, and CSS.
- Use semantic HTML and accessible CSS.

## Design Guidelines
- Keep the layout clean and mobile-friendly.
- Use readable fonts, clear headings, and a simple color palette.
- Make the site easy to navigate with a header and sections.

## User Persona
- College student building a portfolio to showcase academic projects and programming skills.
- Comfortable with Java and Python, learning front-end basics.
- Wants a professional yet approachable design.
